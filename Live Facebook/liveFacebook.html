Encontrando Os Clusters¶

Aqui tentaremos encontrar o nmero de clusters para as 3 melhores variveis, acima vimos que possvel identificar 4 clusters, porm iremos iterar sobre um valor de 1 a 9, e ver como se comprota a mtrica de inertia em cada um desses valores, para determinar o nmero de clusters timos.

In [27]:
from sklearn.model_selection import train_test_split
from sklearn.cluster import KMeans
In [28]:
inertia_values = []
ks = range(1, 10)
for n in ks:
    kmeans = KMeans(n_clusters=n, algorithm='elkan', random_state=42).fit(PCA_components.iloc[:,:3])
    inertia_values.append(kmeans.inertia_)
In [29]:

plt.show()

Inertica uma mtrica que mede a coerncia dos clusters, na figura vemos que aps 4 clusters, a variao entre os valores mnima, ou seja a distoro entre os clusters mnimas, indicando que seja o k timo 4.
